## ✨ Key Features

### 🌍 Global Intelligence
- Real-time city & coordinate search  
- OpenWeatherMap Geocoding API integration  
- Fast & accurate global results  

### 🎨 Immersive UI/UX
- Modern Glassmorphism design  
- Frosted-glass UI components  
- Smooth transitions & hover effects  

### 🌈 Adaptive Theme Engine
- Background changes based on:
  - Clear ☀️
  - Clouds ☁️
  - Rain 🌧️
  - Day / Night cycle 🌙  

### 🔠 Fluid Typography
- Uses `CSS clamp()` for scalable temperature display  
- Fully responsive across devices  

### 🌡️ Environmental Metrics
- Temperature
- Air Quality Index (AQI)
- Humidity
- Wind Speed
- Sunrise & Sunset tracking  

### 📅 5-Day Forecast
- Animated forecast cards  
- Clean daily summaries  
- Staggered animation effects  

### ⚡ Optimized Performance
- 60FPS animations  
- Hardware accelerated CSS transitions  
- Clean AJAX requests  
- Lightweight & fast loading

## 🚀 Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/omsml/WeatherForcast.git
```

### 2️⃣ Get API Key

- Sign up at https://openweathermap.org/
- Generate your free API key

### 3️⃣ Configure API Key

Open `app.js` and replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your actual API key.

### 4️⃣ Run Project

Simply open:

```
index.html
```

in any modern browser.

## 📱 Responsiveness

SkyView follows a **Mobile-First Design Philosophy**:

- 📲 320px – 480px → Optimized mobile layout  
- 📟 640px – 1024px → Tablet grid adjustment  
- 🖥️ 1440px+ → Balanced desktop UI  
- 🖥️ 4K Screens → Fluid typography scaling
